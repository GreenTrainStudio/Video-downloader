# M3U8 Auto Finder (Browser Extension)

Расширение автоматически ищет `.m3u8` ссылки на открытой странице и в сетевых запросах вкладки, после чего показывает найденные ссылки в popup и позволяет скачать каждую по отдельности или все сразу.

## Возможности

- Автопоиск `.m3u8` в HTML/JS контенте страницы.
- Отслеживание `.m3u8` в сетевых запросах вкладки.
- Индикатор количества найденных ссылок на иконке расширения.
- Скачивание одной ссылки или всех найденных ссылок.

## Установка (Chrome / Chromium / Edge)

1. Откройте страницу расширений:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. Включите **Режим разработчика**.
3. Нажмите **Load unpacked / Загрузить распакованное**.
4. Выберите папку проекта.

## Использование

1. Откройте страницу с видеопотоком.
2. Дождитесь, пока страница загрузится и начнутся сетевые запросы.
3. Нажмите на иконку расширения.
4. Нажмите **Скачать** напротив ссылки или **Скачать все**.

## Ограничения

- Некоторые сайты могут скрывать источники через DRM/шифрование; в этом случае прямых `.m3u8` ссылок может не быть.
- Корректность скачивания зависит от доступности URL и CORS/политик сайта.
