# M3U8 Auto Finder (Browser Extension)

Расширение автоматически ищет `.m3u8` ссылки на открытой странице и в сетевых запросах вкладки, после чего отправляет выбранный поток напрямую в **стандартные загрузки браузера**.

## Возможности

- Автопоиск `.m3u8` в HTML/JS контенте страницы.
- Отслеживание `.m3u8` в сетевых запросах вкладки.
- Индикатор количества найденных плейлистов на иконке расширения.
- Запуск загрузки через менеджер загрузок браузера (а не сборка внутри расширения).
- Для master playlist автоматически выбирается вариант с самым большим `BANDWIDTH`.

## Установка (Chrome / Chromium / Edge)

1. Откройте страницу расширений:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. Включите **Режим разработчика**.
3. Нажмите **Load unpacked / Загрузить распакованное**.
4. Выберите папку проекта.

## Использование

1. Откройте страницу с видеопотоком.
2. Дождитесь загрузки страницы и сетевых запросов.
3. Нажмите на иконку расширения.
4. Нажмите **Скачать в браузер** напротив нужного плейлиста или **Скачать все в браузер**.

## Ограничения

- Расширение инициирует загрузку потока в браузере и не перекодирует видео.
- Формат итогового файла зависит от ответа сервера (часто это `.m3u8` playlist/stream URL).
- На некоторых сайтах поток может быть недоступен из-за авторизации, токенов или ограничений сервера.
