# M3U8 Auto Finder (Browser Extension)

Расширение автоматически ищет `.m3u8` ссылки на открытой странице и в сетевых запросах вкладки, после чего позволяет скачать уже **само видео** (собирает `.ts` файл из сегментов плейлиста).

## Возможности

- Автопоиск `.m3u8` в HTML/JS контенте страницы.
- Отслеживание `.m3u8` в сетевых запросах вкладки.
- Индикатор количества найденных плейлистов на иконке расширения.
- Сборка видео из сегментов `.m3u8` и скачивание как `.ts`.
- Скачивание одного видео или всех найденных видео.

## Установка (Chrome / Chromium / Edge)

1. Откройте страницу расширений:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. Включите **Режим разработчика**.
3. Нажмите **Load unpacked / Загрузить распакованное**.
4. Выберите папку проекта.

## Использование

1. Откройте страницу с видеопотоком.
2. Дождитесь загрузки страницы и сетевых запросов.
3. Нажмите на иконку расширения.
4. Нажмите **Скачать видео** напротив нужного плейлиста или **Скачать все видео**.

## Ограничения

- Поддерживается сборка нешифрованных HLS-плейлистов (`EXT-X-KEY` не поддерживается).
- На некоторых сайтах сегменты могут быть недоступны из-за авторизации, токенов или CORS-ограничений.
- Результат сохраняется в контейнере `.ts` (без перекодирования в `.mp4`).
